# 🕷 مستخرج الويب الذكي — Smart Web Scraper

إضافة Google Chrome لإنشاء روبوت استخراج بيانات (Scraper) من أي موقع **دون كتابة أي كود** — بنفس فلسفة إضافة Web Scraper الشهيرة، لكن بواجهة عربية كاملة ولوحة تحكم تظهر داخل الصفحة نفسها.

---

## ✨ القدرات

| القدرة | التفاصيل |
|---|---|
| 🎯 التقاط بصري | انقر على أي عنصر في الصفحة لالتقاطه، مع إبراز وعدّ فوري للعناصر المطابقة |
| 🧠 تعميم ذكي | انقر عنصرين متشابهين فيولّد تلقائيًا محددًا يطابق كل العناصر المشابهة |
| 📄 صفحة أو آلاف الصفحات | استخرج من الصفحة الحالية أو تجوّل تلقائيًا عبر مئات الصفحات |
| 🧭 الترقيم التلقائي | رابط «التالي» (href)، أو زر «عرض المزيد» (AJAX)، أو نمط رابط `?page={page}` |
| ♾ التمرير اللانهائي | يمرّر تلقائيًا حتى يتوقف الموقع عن تحميل محتوى جديد |
| 📊 أنواع البيانات | نصوص، روابط، صور (روابطها)، جداول HTML كاملة، أي سمة (attribute)، HTML خام |
| 📑 جدول تلقائي | انقر داخل أي جدول فيتحوّل تلقائيًا إلى صفوف وأعمدة جاهزة |
| 💾 مشاريع محفوظة | احفظ إعدادات أي موقع وأعد تشغيلها لاحقًا بنقرة واحدة |
| ⬇ التصدير | CSV (مع BOM لدعم العربية في Excel) و XLSX حقيقي |
| 🚫 إزالة التكرار | حذف الصفوف المكررة تلقائيًا أثناء الجمع |

---

## 📥 التثبيت (خطوة بخطوة)

1. **فكّ الضغط** عن ملف `smart-web-scraper.zip` في مجلد دائم (لا تحذفه لاحقًا).
2. افتح Chrome واكتب في شريط العنوان: `chrome://extensions`
3. فعّل **وضع المطوّر (Developer mode)** من المفتاح أعلى اليسار.
4. اضغط **تحميل غير مُعبأ (Load unpacked)** واختر مجلد `smart-web-scraper`.
5. ستظهر الإضافة 🕷 في شريط الأدوات — اضغط أيقونة الدبوس 📌 لتثبيتها.

> ⚠ ملاحظة: المتصفح لا يسمح للإضافات بالعمل على صفحات `chrome://` ولا سوق Chrome الإلكتروني.

## 🚀 الاستخدام — مثال عملي (استخراج منتجات متجر)

### الخطوة 1: فتح اللوحة
افتح الموقع المستهدف، اضغط أيقونة الإضافة ثم **«فتح لوحة الاستخراج في الصفحة»**. ستظهر لوحة جانبية داخل الصفحة.

### الخطوة 2: تحديد حاوية الصفوف (تبويب 🎯 المحددات)
1. اضغط زر 🎯 بجانب «حاوية الصفوف».
2. انقر على **أول بطاقة منتج** في الصفحة.
3. انقر على **بطاقة منتج ثانية** — سيُعمَّم التحديد تلقائيًا على كل البطاقات (سترى عدد العناصر المطابقة ✔).

> لاستخراج قيمة واحدة من الصفحة (مثل العنوان)، فعّل «بدون حاوية (وضع الأعمدة)».

### الخطوة 3: تحديد الحقول
لكل حقل (الاسم، السعر، الرابط، الصورة…):
1. اضغط «＋ إضافة حقل».
2. اكتب اسم الحقل واختر نوعه (نص / رابط / صورة / سمة / HTML).
3. اضغط 🎯 وانقر على العنصر **داخل البطاقة الأولى** فقط — سيطبَّق على كل الصفوف.
4. اضغط 👁 لمعاينة العناصر المطابقة.

> 📑 **لجداول HTML**: اضغط «جدول تلقائي» وانقر داخل الجدول — تُولَّد الصفوف والأعمدة تلقائيًا بأسماء الترويسات.

### الخطوة 4: تجربة سريعة
اضغط **«🔍 تجربة الاستخراج»** لرؤية أول 5 صفوف والتأكد من صحة المحددات قبل التشغيل الكامل.

### الخطوة 5: التنقل بين الصفحات (تبويب 🧭 التنقل)
اختر ما يناسب الموقع:

| الوضع | متى تستخدمه |
|---|---|
| **بدون** | الصفحة الحالية فقط |
| **رابط "التالي"** | المواقع التقليدية بترقيم صفحات — التقط رابط «التالي» بزر 🎯 |
| **زر "التالي" (AJAX)** | زر «عرض المزيد» الذي يحمّل دون إعادة تحميل الصفحة |
| **تمرير لانهائي** | المواقع التي تُحمّل عند النزول (تويتر، انستغرام…) |
| **نمط رابط** | عناوين مرقّمة مثل `?page={page}` — حدد من/إلى |

يتوقف الاستخراج تلقائيًا عند: اختفاء زر «التالي» أو تعطّله، أو بلوغ الحد الأقصى للصفحات، أو توقف البيانات الجديدة.

### الخطوة 6: التشغيل والتصدير (تبويب 📊 البيانات)
1. اضغط **«▶ بدء الاستخراج»** — ستتنقل الصفحة تلقائيًا وتُجمع البيانات مع عدّاد حي (صفحات/سجلات).
2. يمكنك **⏹ الإيقاف** في أي وقت — ما جُمع يبقى محفوظًا.
3. عاين النتائج في الجدول، ثم اضغط **⬇ تصدير CSV** أو **⬇ تصدير XLSX**.

> 💡 أثناء مهمة متعددة الصفحات، **لا تتفاعل مع التبويب** حتى تنتهي — يمكنك متابعة التقدم من اللوحة.

## 💾 المشاريع المحفوظة

- بعد بناء محددات موقع، اضغط **«💾 حفظ المشروع»**.
- من تبويب «مشاريعي»: **تحميل** الإعدادات، **تشغيلها** مباشرة، أو **حذفها**.
- الإعدادات تُستعاد تلقائيًا حتى بعد التنقل بين الصفحات.

## 🧪 موقع تجريبي للتدريب

مجلد `demo/` يحوي موقعًا تجريبيًا كاملًا. شغّله محليًا:

```bash
cd smart-web-scraper/demo
python3 -m http.server 8000
```

ثم افتح `http://localhost:8000/page1.html` وجرّب:
- `page1.html` → ترقيم بروابط (3 صفحات × 8 منتجات)
- `ajax.html` → زر «عرض المزيد» (AJAX)
- `scroll.html` → تمرير لانهائي (60 عنصرًا)
- `table.html` → جدول HTML (زر «جدول تلقائي»)

## ❓ أسئلة شائعة

**الاستخراج يُرجع صفوفًا فارغة؟**
تحقق بزر 👁 أن المحدد يطابق عناصر فعلية. بعض المواقع تُحمّل ببطء — زد «انتظار تحميل الصفحة» من الخيارات المتقدمة.

**الصور تظهر روابط فارغة؟**
الإضافة تلتقط `src` و `data-src` و `srcset` تلقائيًا للصور الكسولة (lazy-load). إن لم تُلتقط، استخدم نوع «سمة» مع `data-src`.

**زر «التالي» موجود لكن الاستخراج توقف بعد صفحة؟**
غالبًا الزر بلا رابط href حقيقي — بدّل الوضع إلى «زر التالي (AJAX)».

**كم عدد السجلات الممكن تخزينه؟**
التخزين المحلي يتسع لعشرات الآلاف من الصفوف النصية. للكميات الضخمة جدًا، صدّر CSV أولًا بأول.

**هل الإضافة ترسل بياناتي لأي خادم؟**
لا. كل شيء يعمل محليًا في متصفحك بالكامل — لا توجد أي اتصالات خارجية.
